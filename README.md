Projet réalisé en Symfony 6 - API platform 3.1 et Vue 3.


## Documentation

[Documentation](https://task-app-sigma-ten.vercel.app/)


## Installation

Lancement de l'API avec docker

```bash
  docker compose up -d
```

Lancement de la partie client

```bash
  cd client && npm i && npm run dev
```
